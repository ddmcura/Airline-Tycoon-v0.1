💸 Daily Tick Logic

Each game day:

Calculate passengers = base demand × random factor

Calculate revenue = passengers × ticket_price

Subtract operating cost

Update aircraft hours & financial stats

Net profit added to cash_on_hand



---

💰 Ticket Price Formula

Base Economy Fare (2020): $0.12 per km

Multipliers:

Business Class: ×2.2

First Class: ×5.0



✅ Sample:
A 1,000 km route =

Economy: $120

Business: $264

First: $600


✅ Fare evolves via inflation curve system (future feature).


---

📈 Demand Formula

Final_Demand = Base_Demand
             × City_Type_Modifier
             × Seasonality_Modifier
             × Reputation_Modifier
             × Price_Multiplier
             × Difficulty_Multiplier

➤ Base Demand Formula

Base_Demand = (Origin_Pop × Destination_Pop) ÷ Distance_Factor

🗓️ Flight Scheduling System (Core Structure)
✅ v1.0 Format (Daily-Only, Simplified):

Each flight is scheduled on a per-aircraft basis

start_time defines when the flight departs

Arrival & turnaround are calculated dynamically

🛩️ Aircraft-Centric Schedule (Primary Source)
json
Copy
Edit
"fleet": {
  "RP-C001": {
    "type": "A320",
    "schedule": {
      "Mon": [
        {"route_id": "MNL-CEB", "start_time": "08:00"},
        {"route_id": "MNL-DVO", "start_time": "14:00"}
      ],
      "Tue": [...]
    }
  }
}
Prevents overbooking

Enables maintenance blocks, downtime

Clean flight rotation logic

🗺️ Route-Centric Mirror (For UI)

Edit

      ]
    }
  }
}
Used for route dashboards

Lets players see “Which planes fly this route, and when?”
